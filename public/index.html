API Rest endpoints disponibles:

Cart

  • POST - localhost:8080/api/carts/
  • POST - localhost:8080/api/carts/:cid/product/:pid/
  • GET - localhost:8080/api/carts/:cid

Products

  • GET - localhost:8080/api/products/
  • GET - localhost:8080/api/products?limit=3
  • GET - localhost:8080/api/products/:pid
  • POST - localhost:8080/api/products - este metodo permite subir varios archivos a la vez con la key photo
  • PUT - localhost:8080/api/products/:pid - este metodo permite subir varios archivos a la vez con la key photo
  • DELETE - localhost:8080/api/products/:pid
